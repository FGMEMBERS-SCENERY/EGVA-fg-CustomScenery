Original README
---------------


This is a first preview of the RAF Fairford (EGVA) custom scenery by J
Maverick 16.

Please put all the files in the correct folder of your FlightGear.

The files into the directory "Models" need to be added into C:\Program Files\FlightGear\data\Models .

For more info feel free to contact me on the forums or via mail (check my Google+ page).
